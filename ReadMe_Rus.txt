Программа zzImage Viewer была написана в целях исследования возможностей библиотеки wxWidgest для python 2.6. Программа позволяет просматривать  изображения в форматах jpg, png, bmp, tiff. Делать выборку из файлов и передавать эту выборку для обработки другим программам. Программы для обработки изображений могут написаны на питоне или других языках и добавлены в папку с плагинами. Программа в процессе загрузки изображений делает миниатюры, для последующий быстрой загрузки. Программа также использует многопоточность, во время загрузки изображений, пользователь, может перемещаться по уже загруженным и просматривать их.
Миниатюры записываются в отдельную папку, Данные о всех файлах изображений записываются в формате XML в FilesBD.XML.

Программа предоставляется как есть, дальнейщих доработок автором не планируется.

Требования:
python 2.6.1
wxPython 2.8
